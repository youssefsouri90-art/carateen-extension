# Carateen CloudStream extension source

هذا مشروع جاهز للبناء على قالب CloudStream.

## البناء
- Windows: `.\gradlew.bat CarateenProvider:make`
- Linux/macOS: `./gradlew CarateenProvider:make`

## ملاحظات
- هذه النسخة تعتمد على سحب روابط الفيديو من صفحة المشاهدة مباشرة بدل الـ API القديم.
- قد تحتاج selectors إضافية إذا غيّر الموقع بنية HTML لاحقًا.
